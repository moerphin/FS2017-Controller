# FS2017-Controller
This is a simple button box for FS2017, based on Arduino Leonardo with 17 joystick buttons. (Switch '0' turns the backlight.)<br>
Components:<br>
-Arduino sketch<br>
-PCB design (eagle)<br>
-BOM<br>
-Autocad design (dwg) for the top of the box<br>
